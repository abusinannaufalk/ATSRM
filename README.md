#  **ATS RESUME MONITOR**
A Python-based resume screening tool that analyzes resumes using **NLP**, extracts skills, and generates an ATS compatibility score.

---

##  Overview

This project reads PDF or text resumes, processes them using NLP techniques, identifies missing keywords, and generates an ATS score to help applicants improve job matching.

---

##  Features

* Resume text extraction
* Keyword detection & scoring
* NLP-based skill analysis
* ATS score generation
* Automated reporting

---

##  Tech Stack

* **Python**
* **NLTK / SpaCy**
* **PyPDF2**
* **Pandas**
